**Gatherly**

Gatherly is a vibrant, campus-focused platform designed to unify event discovery, peer networking, and personal growth tracking for college communities. It enables students to create personalized profiles showcasing their skills, interests, goals, and availability. Through an intuitive interactive campus map, a curated event calendar, and real-time group collaboration spaces, Gatherly fosters meaningful connections and continuous learning within campuses.

Features

Personalized student profiles with skills, interests, and goals

Rule-based matchmaking recommending collaborators with shared or complementary skills

Verified campus event calendar with RSVP functionality

Real-time chat-enabled group collaboration spaces

Gamified growth dashboard tracking student participation and achievements

Interactive campus map for friendly navigation
